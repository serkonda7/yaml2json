# YAML to JSON
[![CI][ci-badge]][ci-workflow]

Utility script for converting YAML files to JSON.

## Installation
```sh
npm i @serkonda7/yaml2json
```

## Usage
```json
// in package.json
"scripts": {
  "convert-yaml": "yaml2json myfile.yml otherfile.yml"
}
```
